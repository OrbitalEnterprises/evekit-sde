This site provides REST access to version ${enterprises.orbital.evekit.sde.release} of the EVE Online Static Data Export (SDE). The end points for the SDE are annotated via Swagger. The Swagger configuration can be found here: ${enterprises.orbital.evekit.sde.ws.basepath}/${enterprises.orbital.evekit.sde.ws.appname}/api/swagger.json .

You can use the Swagger UI demo to explore these endpoints. You can use the Swagger Editor to generate a client for the SDE in the language of your choice (import the URL listed above). See the project page for more details.
